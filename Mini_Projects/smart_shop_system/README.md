# Smart Shop & Login System

## 📌 Description

A beginner-friendly Python project that simulates a login and shopping system.

## 🚀 Features

- User validation using membership operators
- Login system using logical operators
- Age restriction check
- Shopping cart calculation
- Discount logic
- Even/odd bill check

## 🧠 Concepts Used

- Arithmetic Operators
- Comparison Operators
- Logical Operators
- Assignment Operators
- Membership Operators
- Identity Operators
