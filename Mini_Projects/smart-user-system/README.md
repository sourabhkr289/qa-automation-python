# Smart User Validation System 🧠⚙️

## 📌 Overview

This project is a beginner-friendly Python system that simulates real-world user interaction and decision-making.

It covers all foundational programming concepts including variables, data types, operators, conditions, and loops.

## 🎯 Features

- User input handling
- Even/Odd check
- Positive/Negative check
- Largest number finder
- Simple calculator
- Login validation
- Loop-based menu system

## 🛠️ Concepts Covered

- Variables
- Data Types (int, str, bool)
- Operators (arithmetic, comparison, logical)
- Conditional Statements
- Loops (while)

## 🚀 How to Run

Run the main file:

python main.py

## 🧠 Learning Outcome

After completing this project, you will understand how to build a complete logical system using core Python concepts.
